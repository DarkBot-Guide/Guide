# Jak zainstalować i uruchomić DarkBota w Linuksie?

## 1. Instalacja DarkBota na własnym komputerze
- utwórz nowego użytkownika (nie używaj "root");
- otwórz terminal;
- skopiuj-wklej tą komendę i kliknij enter:
> wget -P Downloads https://github.com/0dayYakuza/darkbot_linux/archive/refs/heads/main.zip && cd Downloads && unzip main.zip && cd darkbot_linux-main && sh install_desktop.sh

## 2. Instalacja DarkBota na VPS
- pobierz PuTTy lub inną podobną aplikację i połącz się z VPS;
- utwórz nowego użytkownika (nie używaj "root");
- skopiuj-wklej tą komendę i kliknij enter:
> wget -P Downloads https://github.com/0dayYakuza/darkbot_linux/archive/refs/heads/main.zip && cd Downloads && unzip main.zip && cd darkbot_linux-main && sh install_vps.sh
- Po zakończeniu instalacji zamknij terminal i połącz się z VPS poprzez Remote Desktop Connection (RDP).

## 3. Aby uruchomić darkbota, wpisz polecenie w terminalu:
- cd Downloads/darkbot && java -jar DarkBot.jar.

## Rozwiązywanie problemów
1. Jeśli bot poprosi o weryfikację, otwórz przeglądarkę ręcznie, przejdź do http://localhost:55230/ i autoryzuj swoje konto discord.
2. Jeśli API nie załaduje się, ustaw API na tanos_api w ustawieniach bota lub usuń pliki konfiguracyjne z folderu bota, a następnie spróbuj ponownie.
3. nieznane polecenie wget
- sudo yum install wget
4. nieznana komenda unzip
- sudo yum install uzip
5. AppImages wymagają FUSE do uruchomienia.
- sudo apt install fuse -y && sudo apt install libfuse2 -y
