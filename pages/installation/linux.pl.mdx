# Jak zainstalować i uruchomić DarkBota w systemie Linux?

## 1. Instalacja DarkBota na własnym komputerze
1. utwórz nowego użytkownika (nie używaj „root”);
2. otwórz terminal, skopiuj i wklej this.command i kliknij enter:
`wget -P Downloads https://github.com/0dayYakuza/darkbot_linux/archive/refs/heads/main.zip && cd Downloads && unzip main.zip && cd darkbot_linux-main && sh install_desktop.sh`

## 2. Instalacja DarkBota na VPS
1. pobierz PuTTy lub inną podobną aplikację i połącz się z VPS;

`Link PuTTy:` https://www.putty.org/

2. utwórz nowego użytkownika (nie używaj „root”);
3. otwórz terminal, skopiuj i wklej this.command i kliknij enter:
`wget -P Downloads https://github.com/0dayYakuza/darkbot_linux/archive/refs/heads/main.zip && cd Downloads && unzip main.zip && cd darkbot_linux-main && sh install_desktop.sh`
4. Po zakończeniu instalacji zamknij terminal i połącz się z VPS przez Remote Desktop Connection (RDP).

## 3. Jak uruchomić DarkBota?

Otwórz terminal i wpisz następną komendę:
```bash
cd Downloads/darkbot && java -jar DarkBot.jar
```

## 4. Rozwiązywanie problemów
1. Jeśli bot poprosi o weryfikację, otwórz przeglądarkę ręcznie, przejdź do http://localhost:55230/ i autoryzuj swoje konto na discordzie.
2. Jeśli API się nie ładuje, ustaw API na tanos_api w ustawieniach bota lub usuń pliki konfiguracyjne z folderu bota, a następnie spróbuj ponownie.
3. nieznane polecenie wget
- sudo mniam zainstaluj wget
4. nieznane polecenie rozpakuj
- sudo mniam zainstaluj uzip
5. AppImages wymagają FUSE do działania.
- sudo apt install fuse -y && sudo apt install libfuse2 -y

## 5. Często zadawane pytania
1. install_desktop.sh to:
- aktualizacja systemu i instalacja java 17;
- pobieranie darkbot.zip i rozpakowywanie wszystkich plików oraz usuwanie archiwów.

2. install_vps.sh to:
- aktualizacja systemu i instalacja java 17;
- instalacja i konfiguracja GUI i xRDP;
- pobieranie darkbot.zip i rozpakowywanie wszystkich plików oraz usuwanie archiwów.
