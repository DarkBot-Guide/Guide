# Операционные системы
**1. Windows**
- Windows 7, Windows 8 (8.1), Windows 10, Windows 11
- Windows Server 2019 or 2022

Вам нужна система x64. Бот **НЕ** будет работать, если ваша система 32-битная!

**2. Linux**
- Alma Linux 8 (9)
- CentOS Stream 9
- Fedora 36 (37)
- Rocky Linux 9
- Ubuntu 20.04 (22.04)
- Linux Mint
- Debian 10 (11)

Вам нужна система amd64 (x86_64). Бот **НЕ** будет работать, если ваша система — arm64!

**3. MacOS**
- Нет поддержки MacOS.
- Используйте bootcamp, виртуальную машину или виртуальный частный сервер (VPS).
