# Поддерживаемые операционные системы

**1. Windows**
- Windows 7, Windows 8, Windows 8.1, Windows 10, Windows 11
- Windows Server 2019 и 2022

Ваша система должна быть x64 (64-битная). Бот **НЕ** будет работать, если ваша система x32 (32-битная)!

**2. Linux**
- Alma Linux 8 и выше
- CentOS Stream 8 и выше
- Fedora 36 и выше
- Rocky Linux 8 и выше
- Ubuntu 20.04 и выше
- Linux Mint
- Debian 10 и выше

Ваша система должна быть amd64 (x86_64). Бот **НЕ** будет работать, если ваша система — arm64!

**3. MacOS**
- Нет поддержки MacOS.
- Используйте BootCamp, виртуальную машину или виртуальный частный сервер (VPS).
