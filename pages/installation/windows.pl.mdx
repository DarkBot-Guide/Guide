# How to install & run DarkBot in Windows?

## 1. Instalacja wymaganych aplikacji
1. Pobierz i zainstaluj nasz runtime: https://darkbot.eu/downloads/Runtimes4DarkBot.exe (uruchom go z uprawnieniami administratora)
- Runtimes4DarkBot instaluje Javę 11 i vc++ 2015-2019
2. Możesz zainstalować go ręcznie:
> https://aka.ms/vs/17/release/vc_redist.x64.exe - Visual Studio 2015-2022

> https://download.oracle.com/java/17/latest/jdk-17_windows-x64_bin.exe - Java 17

Jeśli masz jakąś inną wersję Javy, to **skasuj** ją.

## 2. Instalacja DarkBot.jar.
1. Jeśli jesteś **Donor**: Wejdź na stronę #experimental-downloads i pobierz najnowszą wersję bota.
2. Jeśli jesteś **nieDonatorem**:  Przejdź do #unstable-downloads i pobierz najnowszą wersję bota.
3. Umieść pobrany plik DarkBot.jar w swoim folderze z botami.
4. Usuń stary plik DarkBot.exe (jeśli go masz) i kliknij dwukrotnie na DarkBot.jar, aby uruchomić bota.

> Tutorial Video: https://www.youtube.com/watch?v=TwDYbtiDyjA

5. Te same działania, jeśli używasz VPS, ale musisz również wyłączyć ustawienia bezpieczeństwa IE

![DisablingIE](public/vps_ieDisable.png)

## 3. Konfiguracja w grze
 1. Włącz podwójne kliknięcie do ataku
 2. Włącz Start & Stop attack using shortcut bar keys
 3. Włącz auto-logowanie
 4. Upewnij się, że prawidłowo ustawiłeś keybinds
 5. Ustaw grafikę na low i 2D Mode, jeśli masz wolny (stary) PC / VPS.
