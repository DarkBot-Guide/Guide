# System operacyjny
**1. Windows**
- Windows 7, Windows 8 (8.1), Windows 10, Windows 11
- Windows Server 2019 lub 2022

Potrzebujesz systemu x64. Bot **nie będzie działać** jeśli twój system jest 32 bitowy!

**2. Linux**
- Alma Linux 8 (9)
- CentOS Stream 9
- Fedora 36 (37)
- Rocky Linux 9
- Ubuntu 20.04 (22.04)
- Linux Mint
- Debian 10 (11)

Potrzebujesz systemu amd64 (x86_64). Bot nie będzie działał, jeśli twój system to arm64!

**3. MacOS**
- Brak wsparcia dla MacOS.
- Użyj bootcampa, maszyny wirtualnej lub wirtualnego serwera prywatnego (VPS).
