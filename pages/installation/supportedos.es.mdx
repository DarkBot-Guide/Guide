# Sistema operativo
**1. Windows**
- Windows 7, Windows 8 (8.1), Windows 10, Windows 11
- Windows Server 2019 or 2022

Necesitar un sistema basado x64 system. ¡El bot no funcionará si es de 32 bits!

**2. Linux**
- Alma Linux 8 (9)
- CentOS Stream 9
- Fedora 36 (37)
- Rocky Linux 9
- Ubuntu 20.04 (22.04)
- Linux Mint
- Debian 10 (11)

Necesitas un sistema amd64. ¡**El bot no funcionará** si es arm64!

**3. MacOS**
- No soporta MacOS.
- Usa una máquina virtual o un VPS.
