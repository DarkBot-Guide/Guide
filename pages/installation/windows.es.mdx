# ¿Cómo instalar y ejecutar DarkBot en Windows?

## 1. Instalación de aplicaciones requeridas
1. Descargue e instale nuestro tiempo de ejecución: https://darkbot.eu/downloads/Runtimes4DarkBot.exe (ejecútelo con privilegios de administrador)

> Runtimes4DarkBot instalando Java 11 y vc++ 2015-2019.

> Si obtiene este error: "Código de error: 1638 (0x00000666)", significa que el programa ya está instalado.

2. Puedes instalarlo manualmente:
> https://aka.ms/vs/17/release/vc_redist.x64.exe - Visual Studio 2015-2022

> https://download.oracle.com/java/17/latest/jdk-17_windows-x64_bin.exe - Java 17

El bot solo funciona si tiene Java 11 o 17. Si tiene Java 8 u otra versión, **elimínelo**.

## 2. Instalación de DarkBot.jar
1. Si eres **Donante**: Ve a #experimental-downloads y descarga la última versión del bot.
2. Si **no es donante**: Vaya a #unstable-downloads y descargue la última versión del bot.

![botupdate](public/botupdate.png)

3. Coloque el archivo DarkBot.jar descargado en la carpeta de su bot. Todos los archivos se crearán cuando ejecute DarkBot.jar

![botFolder](public/botFolder.png)

4. Las mismas acciones si está utilizando VPS pero también debe deshabilitar la configuración de seguridad de IE

![DisablingIE](public/vps_ieDisable.png)

## 3. Primera ejecución

Ejecute el darkbot haciendo doble clic en el archivo Darkbot.jar. Cuando lo haga, aparecerá esta ventana de verificación:

![firstRun](public/discordAuth.png)

❌ NO CIERRES ESTA VENTANA (Discord Auth) ❌

Esta autenticación de Discord verifica en qué servidores de Discord te encuentras, si estás en un servidor que está en la lista negra del bot, no podrás usarlo.
También comprueba el rol que tienes en nuestro servidor de discordia. Haga clic en autorizar para completar la Autorización.

Si obtenemos esta página, significa que hemos podido autenticarnos con éxito.
Ahora inicia sesión en el bot.

![firstRun](public/discordAuth2.png)

## 4. Iniciar sesión

Al ejecutar el bot nos aparecerá esta ventana para poner los datos de nuestra cuenta.

**1. Contraseña de usuario**

Aquí debe completar su nombre de usuario y contraseña de su cuenta DarkOrbit.

![login](public/login1.png)

**2. Inicio de sesión SID**

Aquí debe usar SID y servidor.

![login](public/login2.png)

¿Cómo encontrar SID y servidor?
1. Haga clic derecho en la página cuando haya iniciado sesión a través del navegador
2. Elija "mostrar sitio / fuente del sitio web" (lo llevará a una nueva pestaña o bloc de notas, etc.)
3. CTRL + F y escribe "sid" y tienes tu sid

![dosid](public/dosid.png)

**3. Salvado**

Aquí podemos agregar y guardar nuestro nombre de usuario y contraseña, por lo que no es necesario completarlo cada vez.

![login](public/login3.png)

## 5. Configuración en el juego
  1. Habilitar doble clic para atacar
  2. Habilite el ataque Iniciar y detener usando las teclas de la barra de acceso directo
  3. Habilitar el inicio de sesión automático
  4. Asegúrese de configurar las combinaciones de teclas correctamente
  5. Configure los gráficos en modo bajo y 2D, si tiene una PC/VPS lenta (antigua) o ejecuta varios bots
