# Операционни системи
**1. Windows**
- Windows 7, Windows 8 (8.1), Windows 10, Windows 11
- Windows Server 2019 or 2022

Нуждаете се от х64 базирана система. DarkBot **НЯМА ДА РАБОТИ** ако имате 32 битова система !

**2. Linux**
- Alma Linux 8 (9)
- CentOS Stream 9
- Fedora 36 (37)
- Rocky Linux 9
- Ubuntu 20.04 (22.04)
- Linux Mint
- Debian 10 (11)

Нуждаете се от amd64 (x86_64) система. DarkBot **НЯМА ДА РАБОТИ** arm64!! 

**3. MacOS**
- Няма подръжка за  MacOS.
- Използвайте bootcamp, virtual machine или Virtual Private Server (VPS).
