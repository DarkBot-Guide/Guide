## 1. Jak automatycznie uruchomić bota?

1. Otwórz folder z botem i utwórz dwa pliki: *runbot.bat* i *file.properties*.

2. W *file.properties* wprowadź login, hasło i hasło główne (jeśli istnieje).

![fileProperties](public/fileProperties.png)

3. W *runbot.bat* wpisz następującą komendę:

```bash
START javaw -jar DarkBot.jar -start -login file.properties
```

![RunBot](public/RunBot1.png)

4. Jeśli chcesz uruchomić bota z określoną konfiguracją dodaj:

```bash
-config config_name
```

Po wszystkich zmianach powinno to wyglądać tak:

![RunBot](public/RunBot2.png)

## 2. Jak uruchomić wiele botów?

1. Zduplikuj folder ze swoim botem i zmień jego nazwę, jeśli chcesz, i wprowadź szczegóły innego konta.

![multipleBots](public/multipleBots1.png)

2. Uruchom je.

![multipleBots](public/multipleBots2.png)

## 3. Jak mogę znaleźć identyfikator SID i serwer?
1. Kliknij prawym przyciskiem myszy na stronie, gdy jesteś zalogowany do DarkOrbit przez przeglądarkę.

2. Wybierz „wyświetl witrynę/źródło witryny” (spowoduje to przejście do nowej karty lub notatnika itp.).

3. CTRL+F i wpisz "SID".

![dosid](public/dosid.png)
