## 1. How to update bot?

If you're **NOT** a donor:

1. Go to #unstable-downloads
2. Download last bot version and replace into your bot folder
3. Restart bot.

If you're a donor:

1. Go to #experimental-downloads
2. Download last bot version and replace into your bot folder
3. Restart bot.

![botupdate](public/botupdate.png)

## 2. How to install plugins?

1. Go to #plugins channel.

Here you will see info abouts plugins like name, min./supported version, description and link.
![plugins](public/plugins1.png)


2. Download a plugin and replace it into bot folder -> plugins.

![plugins](public/plugins2.gif)

3. Open your bot configuration - plugins. 

If you don't see them, click *Reload*. Now you should see them.

![plugins](public/plugins3.png)
![plugins](public/plugins4.png)

4. To activate and configure plugin, tick it and click button like on a screenshot.

![plugins](public/plugins5.png)

## 3. How to autostart bot?

Open your bot folder and create two files: *runbot.bat* and *file.properties*.

1. In *file.properties* add username, login and master-password (if you have it).

![autostart](public/autostart1.png)

2. In *runbot.bat* add next command:

```bash
    START javaw -jar DarkBot.jar -start -login file.properties
```

If you want to run bot with specific config, then add:

```bash
    -config name_of_config
```
