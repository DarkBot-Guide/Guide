# Operation system
**1. Windows**
- Windows 7, Windows 8 (8.1), Windows 10, Windows 11
- Windows Server 2019 or 2022

You need x64 system. Bot will **NOT** work if your system is 32 bit!

**2. Linux**
- Alma Linux 8 (9)
- CentOS Stream 9
- Fedora 36 (37)
- Rocky Linux 9
- Ubuntu 20.04 (22.04)
- Linux Mint
- Debian 10 (11)

You need amd64 (x86_64) system. Bot will **NOT** work if your system is arm64!

**3. MacOS**
- No support for MacOS.
- Use bootcamp, virtual machine or Virtual Private Server (VPS).
