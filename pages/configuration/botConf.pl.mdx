## 1. Jak automatycznie uruchomić bota?

Otwórz folder bota i utwórz dwa pliki: *runbot.bat* i *file.properties*.

1. W *file.properties* dodaj nazwę użytkownika, login i hasło główne (jeśli je posiadasz).

![fileProperties](public/fileProperties.png)

2. W *runbot.bat* dodaj następną komendę:

```bash
START javaw -jar DarkBot.jar -start -login file.properties
```

![RunBot](public/RunBot1.png)

Jeśli chcesz uruchomić bota z określoną konfiguracją, dodaj:

```bash
-config config_name
```

Czyli po wszystkich zmianach powinno to wyglądać tak:
