# 1. Como ejecutar el bot de forma de que se inicie automáticamente?

Abre la carpeta de tu bot y crea los archivos: *runbot.bat* y *file.properties*.

1. EN *file.properties* agrega tus credenciales y tu master-password(Si es que la tienes).

![fileProperties](public/fileProperties.png)

2. En el archivo *runbot.bat* añade este comando:

```bash
START javaw -jar DarkBot.jar -start -login file.properties
```

![RunBot](public/RunBot1.png)

Si quieres ejecutar el bot con una configuración en especifico, entonces agrega:If you want to run bot with specific config, then add:

```bash
-config config_name
```

Después de todos los cambios debería de verse así:
