## 1. ¿Cómo iniciar automáticamente el bot?

Abra la carpeta de su bot y cree dos archivos: *runbot.bat* y *file.properties*.

1. En *archivo.propiedades* agregue nombre de usuario, inicio de sesión y contraseña maestra (si la tiene).

![fileProperties](public/fileProperties.png)

2. En *runbot.bat* agregue el siguiente comando:

```bash
START javaw -jar DarkBot.jar -start -login file.properties
```

![RunBot](public/RunBot1.png)

Si desea ejecutar bot con una configuración específica, agregue:

```bash
-config config_name
```

Entonces, después de todos los cambios, debería verse así:
