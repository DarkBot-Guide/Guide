# How to install plugins?

## 1. If you have the "Donor" role

1. Go to the #plugins channel.

Here you will see information about plugins.

![plugins](public/plugins1.png)

2. Download the plugin and move it to the folder with the bot -> plugins.

![plugins](public/plugins2.gif)

3. Open *Bot Settings* -> *Plugins*.

If you don't see them, click *Restart*. Now you should see them.

![plugins](public/plugins3.png)

![plugins](public/plugins4.png)

## 2. If you don't have the "Donor" role

1. Go to #support-en

2. Write `-pluginsen` and send a message

3. You will see the plugins available to you

![pluginsNonDonor](public/pluginsNonDonor.png)
