## How to autostart bot?

1. Open your bot folder and create a .txt document

- Add username, password, master-password (if you have it)

![autostart](public/autostart1.png)

- Save it as file.properties

2. Create runbot.bat and add next lines (like on screenshot)

Save it and double click RunBot.bat

![autostart](public/autostart2.png)

If you have any bot config and want to run it, add:

- -config *your config name*
