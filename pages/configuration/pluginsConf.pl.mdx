# Jak zainstalować wtyczki?

## 1. Jeśli pełnisz rolę „Darczyńca”.

1. Przejdź do kanału #plugins.

Tutaj zobaczysz informacje o wtyczkach.

![plugins](public/plugins1.png)

2. Pobierz wtyczkę i przenieś ją do folderu z botem -> plugins.

![plugins](public/plugins2.gif)

3. Otwórz *Ustawienia bota* -> *Wtyczki*.

Jeśli ich nie widzisz, kliknij *Uruchom ponownie*. Teraz powinieneś je zobaczyć.

![plugins](public/plugins3.png)

![plugins](public/plugins4.png)

## 2. Jeśli nie masz roli „Darczyńca”.

1. Przejdź do #support-pl

2. Napisz `-pluginspl` i wyślij wiadomość

3. Zobaczysz dostępne wtyczki

![pluginsNonDonor](public/pluginsNonDonor.png)
