## 1. Как сделать автозапуск бота?

1. Откройте папку с вашим ботом и создайте два файла: *runbot.bat* и *file.properties*.

2. В *file.properties* пропишите логин, пароль и мастер-пароль (если есть).

![fileProperties](public/fileProperties.png)

3. В *runbot.bat* пропишите следующую команду:

```bash
START javaw -jar DarkBot.jar -start -login file.properties
```

![RunBot](public/RunBot1.png)

4. Если вы хотите запустить бота с определенной конфигурацией, добавьте:

```bash
-config config_name
```

После всех изменений это должно выглядеть так:

![RunBot](public/RunBot2.png)

### 2. Внутриигровая конфигурация
Включите двойной щелчок для атаки.
Включите "Начинать/останавливать атаку" с помощью клавиш панели быстрого доступа.
Включите автоматический вход.
Убедитесь, что вы правильно установили сочетания клавиш.
Установите низкую графику и режим 2D, если у вас медленный (старый) ПК/VPS или запущено несколько ботов.
